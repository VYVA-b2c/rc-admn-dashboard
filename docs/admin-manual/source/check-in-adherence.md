# Feature Section: Check-in Adherence

## Purpose

Give admins a weekly view of whether scheduled check-in calls were completed, missed, still waiting for a result, or upcoming.

## When To Use It

Use this when reviewing a client's follow-up reliability, investigating missed calls, or deciding whether a staff member should contact the client manually.

## Step-By-Step Instructions

1. Open a client profile.
2. In the Medication and Check-ins area, select View adherence for check-ins when available.
3. Review the weekly calendar and status totals.
4. Use Previous or Next to move between weeks. Future weeks cannot be advanced past the current week.
5. Select Back to profile to return to the client profile.

## What Admins Can Edit

Admins edit the underlying scheduled check-in setup from the client profile or Check-ins monitoring page. The adherence calendar itself is a read-only status view.

## Expected Result

The page shows status totals and a weekly table using these labels:

- Completed
- Missed
- No result logged
- Upcoming

## Common Mistakes

- If no schedule exists, add a scheduled call before expecting adherence data.
- A past scheduled call without a logged result appears as No result logged.
- Non-voice follow-up activity may not appear here unless it is recorded in the check-in data.

## Related Profile Label

The client profile now shows the latest check-in status next to the last check-in value when the data is available. This helps admins quickly see whether the latest interaction was completed, missed, or otherwise unresolved without opening the weekly adherence page.
